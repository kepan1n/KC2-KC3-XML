# KC2-KC3-XML

Веб-приложение для заполнения полей формы `ON_AKTREZRABP` и генерации XML с валидацией по XSD.

## Что уже реализовано

- Парсинг `ON_AKTREZRABP.xsd` и построение формы (поля + описания из `documentation`)
- Пометка обязательных/опциональных полей
- Возможность отключать опциональные поля
- Сохранение значений по умолчанию между запусками (`data/defaults.json`)
- Автоподтягивание части полей из 1-го листа Excel
- Генерация XML (`windows-1251`) и локальная валидация через XSD
- Скачивание сгенерированного XML

## Запуск

```bash
cd /home/konstantin/KC2-KC3-XML
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8080
```

Открыть в браузере: `http://localhost:8080`

## Важно

Проверка на сайте Диадока может требовать ручную загрузку XML в их форму проверки (из-за защиты от ботов).  
Поэтому в приложении сделана строгая локальная XSD-валидация на основе вашего файла `ON_AKTREZRABP.xsd`.
